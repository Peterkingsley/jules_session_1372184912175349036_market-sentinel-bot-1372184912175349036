# Market Sentinel - AI-Powered Market Intelligence Bot

A persistent, conversational Telegram bot ecosystem tailored for the crypto market.

## Features
- **Persistent Greetings**: Unique and permanent welcome messages generated by AI.
- **Conversational Continuity**: Context-aware replies using Google Gemini.
- **Market Intelligence**: Real-time data from CoinGecko.
- **Automated Market Reports**: Scheduled broadcasts (Mon, Tue, Wed) rewritten in the community's brand voice.
- **AI Personality**: energetic, professional yet conversational brand voice.

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Create a `.env` file with:
   ```
   BOT_TOKEN=your_telegram_bot_token
   GEMINI_API_KEY=your_google_gemini_key
   ```
4. Run `npm start`.

## Commands
- `/start`: Welcome message.
- `/p <coin>`: Get real-time price and 24h change for a coin (e.g., `/p bitcoin`).
- `/setbroadcast`: Sets the current group as the target for scheduled market reports.

## Architecture
- `src/bot.js`: Main entry point.
- `src/ai.js`: Gemini AI integration and brand voice logic.
- `src/crypto.js`: CoinGecko API integration.
- `src/scheduler.js`: Weekly report scheduling.
- `src/commands.js`: Command handlers.
